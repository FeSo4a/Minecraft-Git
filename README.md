# Mcgit - Minecraft 快捷栏存档管理工具

- 版本: v1.0.0
- 作者: FeSo4a
- 许可证: MIT

Mcgit 是一个专为 Minecraft 玩家设计的命令行工具，用于便捷地管理和备份你的物品栏（Hotbar）配置文件。

## 核心特性

- 自动记录配置：首次设置好游戏路径、存档路径和版本号后，程序会自动记录。后续操作时只需输入简单的指令（如 -hb），无需重复输入冗长的路径参数。
- 便捷的备份与恢复：支持一键备份当前版本的 hotbar 文件，或快速恢复到之前的存档状态。
- 跨平台支持：原生 Python 开发，提供 Windows 下打包好的 .exe可执行文件，开箱即用。

## 技术栈

- Python 3.13
- argparse(命令行参数解析)
- os(操作系统路径交互)

## 安装与运行

1. 下载发行版
    前往 Releases页面下载最新版本的 Mcgit v1.0.0打包文件（.exe）。
2. 运行方式
    在命令行中调用：

    ```cmd
    mcgit [选项]
    ```

## 使用指南

### 首次配置

在首次使用时，你需要通过参数指定游戏目录和存档目录：  

```bash
:: 设置游戏版本、游戏路径和保存路径
mcgit -v 版本 -g .minecraft文件夹路径 -s 设置保存文件的路径（默认..\\save）
```

配置完成后，这些信息会被程序自动保存，下次无需再次输入。

### 备份与恢复

配置完成后，日常操作变得极其简单：

```bash
:: 备份当前版本的 hotbar 文件
mcgit -hb

:: 恢复当前版本的 hotbar 文件
mcgit -hs
```

### 高级操作（手动指定路径）

如果你需要临时操作其他路径的文件，依然支持直接传入路径参数：

```bash
:: 将指定文件夹下的 hotbar 文件转移到当前设置的 .minecraft 目录下
mcgit -c "D:\OtherSaves\Backup1"

:: 将当前备份恢复到指定的 .minecraft 目录下
mcgit -r "D:\OtherMinecraft\.minecraft"
```

## 完整参数列表

| 短参数 | 长参数          | 说明                                      | 示例                     |
| :----- | :------------   | :--------------------------------------- | :----------------------- |
| `-h`   | `--help`        | 显示帮助信息并退出                        | `mcgit -h`               |
| `-v`   | `--version`     | **设置**当前游戏版本号                    | `mcgit -v 1.20.1`        |
| `-g`   | `--gamepath`    | **设置** `.minecraft` 文件夹路径          | `mcgit -g "C:\..."`      |
| `-s`   | `--setpath`     | **设置**保存文件（备份）的路径             | `mcgit -s "D:\..."`      |
| `-sh`  | `--showhotbar`  | **显示**保存文件的路径                    | `mcgit -sh`              |
| `-sp`  | `--showpath`    | **显示**目前指向的 `.minecraft` 路径      | `mcgit -sp`              |
| `-sv`  | `--showversion` | **显示**目前指向的版本号                  | `mcgit -sv`              |
| `-hb`  | `--hotbarbackup`| **备份**当前版本的 hotbar 文件            | `mcgit -hb`              |
| `-hs`  | `--hotbarstore` | **恢复**当前版本的 hotbar 文件            | `mcgit -hs`              |
| `-c`   | `--copy`        | 将指定位置的 hotbar 文件**复制**到游戏目录 | `mcgit -c "D:\..."`      |
| `-r`   | `--restore`     | 将备份文件**恢复**到指定游戏目录           | `mcgit -r "D:\..."`      |

## 注意事项

- 文件位置：

    ```path
    空文件夹/
        mcgit/
            mcfig.exe
            _internal/
                ...
        save/   <- 程序会在这里自动创建一个文件夹
            ...
        config/
            config.json   <- 配置文件，同样也会自动创建
    ```

- 路径格式：Windows 系统中建议使用双引号 "包裹包含空格的文件路径。
- 首次运行：必须先运行一次带有 -g和 -s参数的命令来完成初始化配置。
- 文件覆盖：在进行恢复 (-hs, -r) 或复制 (-c) 操作时，若目标位置已存在同名文件，将被直接覆盖，请谨慎操作。
